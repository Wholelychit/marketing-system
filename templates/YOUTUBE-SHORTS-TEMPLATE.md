# YouTube Shorts Template

## Hook

Start with a simple emotional problem.

Examples:

- Reading feels overwhelming.
- Math does not have to feel scary.
- Slow learners can still succeed.

## Main Message

Keep it:

- short
- calm
- practical
- emotionally safe

## Core Positioning

- 30 minutes a day
- one lesson at a time
- small daily progress

## Call To Action

Examples:

- Try one lesson today.
- Start small.
- Build confidence slowly.
- Practice for 30 minutes today.
