13. PageSpeed Insights testing.
14. Schema markup testing.
15. Canva folder for brand graphics and social assets.
16. Social posting plan for YouTube, Facebook, Instagram, TikTok, Pinterest, Reddit, LinkedIn, and X.

## Performance Rules

Use these rules on all website repositories:

1. Mobile first.
2. Simple HTML/CSS/JS unless the site truly needs a framework.
3. No unnecessary animations, scripts, popups, or trackers.
4. Compress images before upload.
5. Use clear page titles and meta descriptions.
6. Add internal links between related pages.
7. Keep pages readable at a 7th to 9th grade level for consumer sites.
8. Test home page, mobile layout, navigation, and contact paths after each major change.

## Do Not Repeat Work Rule

Before editing any site, check:

1. README.
2. AGENT-INSTRUCTIONS.
3. Latest stable checkpoint.
4. Current live page.
5. The exact file being changed.
