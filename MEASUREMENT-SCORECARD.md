# Measurement Scorecard

This file defines what to measure so marketing work can improve instead of guessing.

## Measurement Rule

Measure only what helps the next decision.

Do not chase every number.

## Required Free Tools

| Tool | Purpose | Cost |
|---|---|---|
| Google Search Console | Search keywords, impressions, clicks, page indexing | Free |
| Google Analytics 4 | Visits, engagement, traffic sources, conversions | Free |
| Microsoft Clarity | Scroll depth, dead clicks, rage clicks, recordings | Free |
| GitHub | Track shipped files and commits | Free |
| Native social analytics | Views, saves, shares, comments | Free |

## Core Metrics by Funnel Stage

| Stage | Metric | Meaning |
|---|---|---|
| Visibility | Impressions | People saw the page/post in search or social |
| Interest | Clicks / views | People chose to look |
| Engagement | Scroll depth / time / saves | People cared enough to continue |
| Action | CTA clicks / starts / leads | People took the desired next step |
| Improvement | Return visitors / repeat starts | People found enough value to come back |

## Result Grades

| Grade | Meaning | Action |
|---|---|---|
| A | Clear winner | Improve, expand, and repurpose |
| B | Working | Keep going and test one change |
| C | Weak but possible | Fix headline, CTA, or page layout |
| D | Not working | Rework topic or audience |
| Blocked | Cannot measure or publish | Fix setup before more production |

## Decision Rules

- If search impressions are high but clicks are low, improve title and meta description.
- If clicks are good but engagement is low, improve page opening and layout.
- If engagement is good but CTA actions are low, improve button text and placement.
- If social views are good, turn the post into a website article.
- If an article gets traffic, turn it into short videos and internal links.
