# SEO System

This file is the reusable SEO process for Wholelychit websites.

## Goal

Create helpful pages that answer real user questions, build trust, and give each website a clean path for long-term search traffic.

## Core SEO Rule

One page should answer one main problem clearly.

Do not stuff keywords.

Do not write thin pages just to rank.

## Page Types

| Page Type | Best For | Example |
|---|---|---|
| Beginner guide | New visitors | How to help a struggling reader |
| Checklist | Quick help | New puppy checklist |
| FAQ page | Repeated questions | Free slots beginner FAQ |
| Comparison page | Decision help | Reading practice vs tutoring |
| Local/service page | Local intent | Restaurant AI assistant for small restaurants |
| Resource page | Long-term trust | Parent reading resources |

## Keyword Process

For every page, choose:

1. Main keyword
2. Two to four related phrases
3. Search intent
4. Reader problem
5. Helpful next step

## Search Intent Types

| Intent | Meaning | Page Style |
|---|---|---|
| Learn | User wants explanation | Guide or FAQ |
| Do | User wants steps | Checklist or how-to |
| Compare | User is choosing | Comparison page |
| Find | User wants a tool/site/service | Landing page |
| Trust | User needs confidence | About, review, proof, safety page |

## SEO Page Template

```md
# [Clear Page Title]

Meta title:
Meta description:
Main keyword:
Secondary keywords:
Reader problem:
Search intent:
CTA:
Internal links:

## Quick Answer

Give the answer in 2-4 simple sentences.

## Why This Matters

Explain the problem in plain language.

## Step-by-Step Help

Use short sections and numbered steps.

## Common Mistakes

List the mistakes and explain how to avoid them.

## Helpful Next Step

Give one clear next action.

## FAQ

Add 3-6 short questions and answers.
```

## Metadata Rules

### Meta Title

Use 45-60 characters when possible.

Good format:

`Main Topic | Site Name`

Examples:

- Calm Reading Practice for Kids | ReadEasy30
- New Puppy Checklist | PetNeeds.ai
- Free Slots for Beginners | SlotsFreeUSA

### Meta Description

Use 140-160 characters when possible.

Good format:

`Simple benefit + who it helps + next step.`

Example:

`A calm 30-day reading practice path for children, parents, and helpers who want simple daily lessons without pressure.`

## Internal Link Rules

Each page should link to:

- Homepage or main app page
- One related guide
- One next-step page
- About or safety/trust page when useful

## Project SEO Priorities

- calm reading help for children
- parent reading support

- Avoid promising a child will reach a grade level by a fixed date.
- Position the site as practice support, not a school replacement.

Trust notes:

### SlotsFreeUSA

Priority topics:

- free slots for beginners
- slot terms explained
- free slot games
- play slots for fun
- responsible gambling basics

Trust notes:

- Be clear that slots involve risk when real money is involved.
- Do not promise wins.
- Keep entertainment-first language.

## Content Quality Checklist

Before publishing an SEO page:

- Main keyword appears naturally in title or intro
- Quick answer appears near the top
- Page solves one problem
- Headings are clear
- Paragraphs are short
- Internal links are useful
- CTA is simple
- No hype
- No unsafe claims
- Mobile reading is easy

## Improvement Cycle

Every month:

1. Check which pages were published.
2. Improve weak titles.
3. Add internal links.
4. Add FAQ sections.
5. Update stale facts.
6. Create one follow-up page for the strongest topic.
