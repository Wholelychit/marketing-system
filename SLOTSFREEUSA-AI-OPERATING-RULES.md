# SlotsFreeUSA AI Operating Rules

Last updated: 2026-05-28

## Core Rule

SlotsFreeUSA works independently.

AI helps Gerry build, update, market, and manage the business, but the public website must not depend on AI to load, rank, or be reviewed.

## Project Contact

Use this public project email for SlotsFreeUSA business, affiliate, review, and visitor contact:

`slotsfreeusa@gmail.com`

## Allowed AI Support Now

AI may help with:

1. Static website updates.
2. SEO page drafts.
3. Beginner-friendly content.
4. Affiliate outreach letters.
5. Review templates.
6. Content calendars.
7. Social posts.
8. YouTube Shorts scripts.
9. Canva prompts.
10. Site review checklists.

## Controlled AI Later

Add public AI features later only when they improve revenue, trust, or visitor clarity.

Possible later uses:

1. Bonus-term explainer using approved source text.
2. Fixed-answer safer-play FAQ.
3. Partner comparison helper using verified review data.
4. Slot-term glossary helper.
5. Internal marketing assistant for Gerry.

## Do Not Add Yet

Do not add:

1. Public open-ended AI chat.
2. Personalization based on hidden tracking.
3. Any tool that implies guaranteed outcomes.
4. Any tool that gives legal, financial, or betting instructions.
5. Any live third-party AI script before privacy review.

## Working Standard

Use AI for speed and management.

Keep the site static, fast, crawlable, affiliate-review-ready, and useful without ChatGPT being open.
