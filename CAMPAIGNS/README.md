# Campaigns

This folder holds reusable marketing campaigns for Wholelychit projects.

## Rules

- Keep campaign language simple and practical.
- Use short sentences for consumer-facing copy.
- Do not promise guaranteed results.
- Do not add private keys, API tokens, live tracking, live ads, payment setup, or framework changes.
- Campaign files can include SEO ideas, social posts, video scripts, Canva asset notes, landing page notes, and manual launch checklists.

## Campaign File Template

Use this structure for future campaign files:

1. Campaign goal.
2. Audience.
3. Simple promise.
4. Website updates needed.
5. SEO topics.
6. Social posts.
7. Short-video scripts.
8. Canva asset notes.
9. Manual launch checklist.
10. Blockers.
