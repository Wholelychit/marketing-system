# Monday Launch One-Page Checklist

Start date: 2026-06-01

Use this first on launch day. It keeps the ReadEasy30 / MathEasy30 promotion simple and prevents jumping between too many files.

## Goal

Start public awareness for ReadEasy30 using calm, no-shame reading-practice language.

## Do Not Start Yet

Do not add or turn on:

- paid ads
- live ad scripts
- tracking pixels
- payment tools
- email signup tools
- auto-posting tools
- accounts or logins
- affiliate links
- API keys

Manual posting only until accounts, privacy, spam handling, platform rules, and approval flow are ready.

### Step 2 — Make one simple Canva image

Use this text:

> Reading practice without shame
> 30 minutes a day
> Beginner to Grade 8 readiness

Design rules:

- large readable text
- calm colors
- no clutter
- mobile-friendly square or vertical version
- include ReadEasy30.com

### Step 3 — Post the first message

Use this as the first general post:

>
> Link: https://readeasy30.com/app.html

Answer these before making more assets:

1. Which post was easiest to create?
2. Which wording felt strongest?
3. Which link should be reused?
4. Did any post get a comment, click, save, or reaction?
5. What should be improved tomorrow?
