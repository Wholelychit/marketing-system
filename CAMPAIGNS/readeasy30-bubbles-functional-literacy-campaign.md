# ReadEasy30 Bubbles + Functional Literacy Campaign

Last updated: 2026-05-29

## Purpose

Use the strongest ReadEasy30 marketing assets:

1. Bubbles as the calm, non-judgmental reading coach.
2. Functional literacy as the practical-life search angle.
3. No-shame reading practice for adult learners, ESL learners, older learners, parents, tutors, and helpers.

## Core campaign idea

Reading practice should feel calm, useful, and private.

ReadEasy30 should not be marketed as another noisy learning app. It should be marketed as a calmer reading-practice path with practical lessons and a friendly guide.

## Main strengths to emphasize

### Bubbles

Bubbles is a major trust and comfort asset.

Use Bubbles as:

- a calm reading coach
- a no-shame helper
- a friendly guide through lessons
- a symbol for slower, safer practice
- a recurring visual mascot for posts, pins, videos, and worksheets

Avoid making Bubbles feel like a noisy game character. Bubbles should feel patient, soft, and encouraging.

### Functional literacy

Functional literacy should be a major SEO and content direction.

High-value topics:

- reading signs
- reading notes
- reading forms
- reading schedules
- reading medicine labels with adult supervision
- reading job notes
- reading school notes
- reading simple instructions
- reading appointment reminders
- reading community notices

This angle can attract adult literacy, ESL, homeschool, parent-helper, and practical reading traffic.

### Simple navigation strengths

Use clear landing-page language around:

- Adult Reading Without Shame
- ESL Reading Practice
- Free Printable Reading Worksheet
- Reading Levels Beginner to 8th-Grade Readiness
- Open Placement Check
- 30-Minute Reading Practice

## Safe claim rules

Do not claim:

- treats dyslexia
- fixes ADHD reading problems
- solves special education needs
- replaces teachers, tutors, schools, or specialists
- guarantees 8th-grade reading ability
- provides official placement or diagnosis
- is clinically proven unless sourced later

Use safer wording:

- may help learners who need calmer practice
- useful for parent/helper-supported reading practice
- designed to reduce pressure and distraction
- short lessons for steady confidence
- practical reading for everyday life
- built for no-shame practice

## Community post strategy

Do not call it infiltration.

Call it community-help posting.

Candidate community types:

- parent groups
- homeschool groups
- ESL groups
- adult literacy groups
- tutor groups
- special education support groups
- dyslexia support groups
- local library/community groups

Before posting:

1. Read the rules.
2. Do not post if promotion is banned.
3. Help first.
4. Link only when allowed or clearly relevant.
5. Be transparent that the site is being built.
6. Ask for feedback instead of pretending it is perfect.

## Reddit comment reply templates

### If someone asks if it is for dyslexia

It is not a dyslexia treatment or diagnostic tool. I am building it as a calm reading-practice resource that may be easier for some struggling readers to use because it avoids pressure, clutter, and shame.

### If someone asks if it is for ADHD

It is not an ADHD treatment. The design goal is simple: short practice, calm screens, fewer distractions, and a clear routine.

### If someone asks if it is free

The current goal is to keep it easy to start and useful as a simple practice resource.

### If someone asks about 8th grade

The long-term goal is to build a path toward stronger reading readiness over time. I avoid calling it official grade placement because it is a practice site, not a school assessment.

### If someone criticizes the site

Thank you. That is useful feedback. I am building this step by step and want it to be practical for real learners and helpers.
