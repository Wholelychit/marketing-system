# Education Canva Production Checklist

Use this checklist to produce the first ReadEasy30 / MathEasy30 launch graphics.

Primary prompt file:

- `CAMPAIGNS/education-canva-asset-prompts.md`

Primary launch file:

- `CAMPAIGNS/monday-promotion-launch.md`

## Rules

- Keep text large and readable.
- Use calm colors.
- Do not clutter the image.
- Do not promise guaranteed results.
- Add `ReadEasy30.com` clearly.
- Use one main idea per image.

## Export Checklist

For every graphic:

- Export PNG.
- Save with clear filename.
- Check text at phone size.
- Confirm URL is visible.
- Confirm no tiny text.
- Confirm no misleading claim.
- Record filename or Canva link in the posting log.
