# SlotsFreeUSA Review Template Examples

Last updated: 2026-05-28

## Purpose

Use these examples when creating future partner review cards or full review pages. Keep monetization links inactive until approval, tracking, age/location rules, terms, disclosure, and privacy impact are checked.

## Status Labels

Use clear labels:

- Pending Review
- Approved Partner
- Paused
- Removed
- Content Only
- Link Not Active

## Template 1: Pending Partner Review

```md
### [Partner Name]

Status: Pending Review
CTA status: Link not active
Best for: [Beginner / mobile users / bonus researchers / social casino users]
Age/location note: Terms must be checked before publishing live link.
Offer summary: Offer details are not published until verified.
What we are checking:
- age rules
- location rules
- bonus terms
- support access
- responsible-play tools
- tracking and disclosure requirements
Visitor note: Do not create an account until you read the official terms.
Disclosure: SlotsFreeUSA may earn compensation if this becomes an approved partner later.
Last reviewed: [date]
```

## Review Page Checklist

Every partner review should include:

- clear status label
- age/location warning
- offer summary
- what to check first
- support note
- safer-play note
- disclosure note
- last reviewed date
- inactive link if not fully approved

## Do Not Publish

Do not publish any review that:

- promises winnings
- suggests gambling as income
- hides disclosure
- claims legal access everywhere
- uses fake review language
- lacks last reviewed date
- lacks age/location warning
