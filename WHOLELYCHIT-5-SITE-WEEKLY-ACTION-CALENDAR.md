# Wholelychit 5-Site Weekly Action Calendar

Last updated: 2026-05-29

Use this calendar to avoid jumping around. Each week has one main site, one support site, and one simple output goal.

## Working Rule

Do not try to fully build all five projects in one day.

Every week should produce:

1. one improved page or draft
2. one social/post batch
3. one Canva prompt batch
4. one status update
5. one next action

## Site Priority Rotation

| Week | Main Site | Support Site | Main Goal |
|---|---|---|---|
| Week 1 | ReadEasy30 | MathEasy30 | Education launch and no-shame practice messaging |
| Week 2 | SlotsFreeUSA | PetNeeds.ai | Trust pages, beginner guides, partner prep |
| Week 3 | PetNeeds.ai | ReadEasy30 | Pet category content and family-friendly traffic |
| Week 4 | RestaurantAIbot / Resturants.ai | SlotsFreeUSA | Local business offer clarity and lead path planning |
| Week 5 | MathEasy30 | ReadEasy30 | Sister-site education cross-promotion |

Repeat the cycle after Week 5.

## Daily 30-Minute Marketing Sprint

Use this when time is short.

1. Pick one site.
2. Pick one page or post.
3. Improve one headline.
4. Add one helpful section.
5. Add one clear next step.
6. Record what changed.

## Monthly Review

At the end of each month, record:

- best page created
- best post created
- weakest site
- strongest site
- biggest blocker
- next month priority
- revenue path that is closest to safe activation

## Safety Checks Before Publishing

### SlotsFreeUSA

- 21+ language present where needed
- no winnings promises
- no unverified state/legal claims
- no live affiliate links without approval
- disclosure present

### RestaurantAIbot / Resturants.ai

- no payment setup
- no ordering integration
- no automated outreach sending
- no fake customer claims
- offer is simple

### ReadEasy30 and MathEasy30

- no shame language
- no medical/diagnostic claims
- no official school-placement claims
- adult guidance encouraged for younger learners
- app links work

### PetNeeds.ai

- no diagnosis claims
- veterinarian guidance included where needed
- product links not live unless approved
- no user photo upload system without privacy plan
