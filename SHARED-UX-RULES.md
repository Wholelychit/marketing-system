# Wholelychit Shared UX Rules

Central UX standard for all Wholelychit websites.

Official shared resource:
https://github.com/Wholelychit/marketing-system/blob/main/SHARED-UX-RULES.md

All repositories should reference and follow this file.

## Core Rules
- Mobile-first
- Plain HTML, CSS, JavaScript preferred
- Fast loading
- Simple navigation
- Warm emotional design
- Readable text
- Consistent headers and footers
- Emotional trust and clarity
- Simple user journeys
- Avoid clutter and confusing layouts

## Required Repository Files
- AGENT-INSTRUCTIONS.md
- UX-RULES.md
- CHANGELOG.md
- sitemap.xml
- robots.txt

## Measurement Standards
All sites should support:
- Microsoft Clarity
- Google Analytics
- Google Search Console

Track:
- bounce rate
- scroll depth
- return visitors
- rage clicks
- dead clicks
- engagement zones
