# SlotsFreeUSA Canva Production Checklist

Last updated: 2026-05-28

## Purpose

Use this checklist when turning SlotsFreeUSA topics into Canva graphics, Pinterest pins, short video covers, and social images.

## Before Creating

Choose one topic:

- Play Free First
- Bonus Terms Checklist
- Slot Terms Explained
- Social Casino Coins
- Sweepstakes Rules
- Responsible Play
- How We Review Sites

Choose one destination page:

- `https://slotsfreeusa.com/play-free-slots.html`

- `https://slotsfreeusa.com/how-we-review.html`

## Design Settings

Use:

- dark navy background
- gold accents
- white readable text
- simple slot-style icons
- large headline
- short subtitle
- clean spacing
- mobile-readable layout

Avoid:

- clutter
- flashing casino chaos
- cash piles
- jackpot promises
- tiny text
- risky claims

## Export Sizes

Create these versions when useful:

| Format | Size |
|---|---|
| Pinterest Pin | 1000 x 1500 |
| Instagram/Facebook Square | 1080 x 1080 |
| Short Video Cover | 1080 x 1920 |
| Website Thumbnail | 1200 x 630 |

## Text Safety Check

Allowed phrases:

- Play free first
- Read terms
- Learn the rules
- Compare carefully
- Stay in control
- Entertainment only
- No deposits here
- No payouts here

Do not use:

- win big
- easy money
- guaranteed payout
- secret system
- risk-free gambling
- legal everywhere
- make money with slots

## Final Check

Before using the graphic:

1. Is the text readable on phone?
2. Is there only one main message?
3. Does it avoid risky claims?
4. Does it point to the correct page?
5. Does it match SlotsFreeUSA navy/gold style?
6. Is the image file name simple?
7. Is the matching post logged in `SLOTSFREEUSA-MANUAL-POSTING-LOG.md`?
